# 程序参数的确认

参数13与参数14，对应不同霍尔，不同机型匹配计算公式表

其中，霍尔量程分400A，900A等为±量程

​	供电电压通常为：5V，零电流偏置为：2.5V。

根据电流灵敏度计算

输出电压Vout = Voffset + Sen * Ip

​	Voffset：零电流偏置电压 Vcc = 5V时，= 2.5V

​	Sen：电压灵敏度，400A - 5mv/A，900A - 2.22mv/A

硬件实际输出电压范围：

​	0.5 ~ 4.5V

因为芯片IO端口的输入电压最大为3.3V所以，硬件电路需要做分压处理

​	4.5V --> 3V

映射到控制板IO口的电压值为：0.33 ~ 3V

| Hall电压   | 0.5 ~  | 2.5V   | ~4.5V  |
| ---------- | :----- | :----- | ------ |
| IO电压     | 0.33 ~ | 1.667V | ~ 3.0V |
| AD采样     | -32768 | 0      | 32767  |
| 采样电流Ip | -Ipmax | 0      | +Ipmax |
