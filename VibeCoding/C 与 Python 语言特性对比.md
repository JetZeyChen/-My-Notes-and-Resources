# C 与 Python 语言特性对比

## 运行环境

​	C 语言是面向底层的高级语言，由编译器通过，.c —预处理––> .i —汇编 —> .s —编译—> .o ——链接—->.out / .elf ；从高级语言到底层语言的生成，最终生成的二进制机器码可以在芯片内部指定好的指令集快速的响应和运行。由于面向底层对象，所以每一个指令都需要是准确无误的，在编译之前就需要确定好数据的类型，即该变量占据的内存空间大小。C 语言需要运行在指定的 Soc 上就需要对应芯片架构的指令集，通过编译器选择对应的编译工具链生成目标机器上可执行的二进制机器码。

​	Python 是面向对象的解释型高级通用编程语言，主要通过解释器环境来对输入的 Python 语言进行解释翻译成 C 语言这样的面向底层的代码才能够真正的在机器上被运行。由于 Python 是解释型的编程语言，所以其并不关心底层的实现，只需要处理顶层的软件逻辑即可。Python 的具有动态类型特性，编程过程无需关系变量实际占用多少内存，直接使用即可，只有在运行过程中才会报类型错误问题。

## 语法结构

### 语句块

C 语言使用 `{}`表示一个完整的语句块

```c
char* C = 'Clang'; //且每条语句都需要以';'结束
{
    int C = 1;
    print('%s is nerver die', C);
    print('%d is nerver die', C);
}
```

Python 使用缩进表示每一个语句块

```py
python = 'python'
if python == 'python is perfect': #':'之后的缩进语句等效于 C 的 '{}'
	return false
```



### 注释

C 语言的注释有两种：

```c
//注释1-单行注释
/*注释2
	多行注释
*/
```

Python 的注释只有一种：

```py
#注释
```
